# The 2025 Canon (archived)

This directory preserves, **verbatim**, the original open-source Memento framework as published in August 2025: the artefact of **Epoch 4, The Memento Framework** (the documentation-discipline era).

It is kept intact deliberately. Memento's central claim is that continuity across hundreds of working sessions (what was tried, what was decided, what was superseded and why) must survive. This archive is that claim applied to the framework itself: the abstraction exactly as it stood before a year of field use outgrew it. The current canon records what changed and why, with receipts.

**Numbering note:** the documents here describe "five evolutionary epochs", numbered 1–5. The 2026 canon renumbers them 0–4 and adds **Epoch 5: Falsifiable Governance**. So "5. MEMENTO Framework" in these pages corresponds to Epoch 4 in the current canon.

Nothing in this directory is maintained. For the living framework, start at the repository root `README.md`.
