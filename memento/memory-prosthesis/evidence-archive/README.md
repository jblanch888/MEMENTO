# evidence-archive

Dated, immutable records: plans (`plan-*`), reference memos (`reference-*`), findings (`finding-*`), handovers (`handover-*`). Once banked, a memo's BODY is history: superseded with a banner or a dated ERRATUM/RESOLUTION/AMENDMENT note, and silent edits are banned (CD #4d). Every plan required by PLANNING §0 lives here as an extant artefact. Imports from the deployment estates carry a de-identification note in their provenance (CD #4e).

**Status semantics (stated 2026-07-21, external-review finding 3):** the frontmatter `status:` field is LIVE — it is maintained at life events (approval, closure, supersession), so a reader can trust it as current. Everything else in a banked memo, including `OPEN` markers, speaks as at banking; later resolutions arrive as dated notes in place or in successor memos. For current truth, start at the working context and the CHARTER, and treat this archive as the receipts behind them.
