# evidence-archive

Dated, immutable records: plans (`plan-*`), reference memos (`reference-*`), findings (`finding-*`), handovers (`handover-*`). Once banked, a memo is history — superseded with a banner or an ERRATUM pointer, never silently edited (CD #4d). Every plan required by PLANNING §0 lives here as an extant artefact. Imports from the deployment estates carry a de-identification note in their provenance (CD #4e).
