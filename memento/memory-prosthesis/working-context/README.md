# working-context

Session-facing state, and nothing else: `CURRENT_FOCUS.md` (mission, active playbook, current task, constraints, next actions) and `STATUS.md` (current-session snapshot). Read on every restart (CD #8). Every edit is a coherent replacement, never an append (CD #11); the pre-compact protocol (CD #9) resets both. No history accumulates here — history lives in the evidence archive.
